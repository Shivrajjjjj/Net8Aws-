About Net8Aws

Net8Aws is a demonstration project showcasing how to build and deploy .NET 8 applications with AWS ECS, CI/CD pipelines, and Dockerized environments.

Features include:

  • Modern .NET 8 backend API
  • Static frontend served via Docker or directly from backend
  • AWS ECS service deployment with CloudFormation
  • CI/CD automation using CodeBuild and CodePipeline
